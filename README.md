# coffeemakesgames.com

Static site for Coffee Makes Games, a solo indie game studio. First game: Dungeon Story (iOS, coming soon).

## Local preview
`python3 -m http.server 8000`

## Deploy
Push to `main`. GitHub Pages serves from root, custom domain via `CNAME`.

DNS A records for `@`: `185.199.108.153`, `185.199.109.153`, `185.199.110.153`, `185.199.111.153`.
CNAME `www` → `<github-username>.github.io`.
