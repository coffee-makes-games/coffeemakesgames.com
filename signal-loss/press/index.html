Short description

SIGNAL//LOSS is a minimalist arcade survival game about keeping a collapsing signal alive.

Long description

Rotate the phase needle, tune drifting signal rings, and hold alignment long enough to sync the system before the signal tears itself apart. Each run starts calm, then quickly collapses into faster rings, hostile glitches, screen noise, and split-second survival decisions.

Tune the rings. Hold the signal. Delay the collapse.

Features

  • Frenetic one-more-run arcade survival.
  • Simple touch controls: steer the needle and hold the signal.
  • Multiple unlockable difficulty modes, from DRIFT to TERMINAL COLLAPSE.
  • Beneficial and hostile glitches that change each run in real time.
  • Archive entries that reveal signal anomalies as you encounter them.
  • Mode-specific leaderboards and best sync tracking.
  • Clean neon pixel-art presentation built around rings, waves, and signal corruption.
